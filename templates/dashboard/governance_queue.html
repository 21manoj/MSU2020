Mass upload — user profiles

UTF-8 CSV with columns username, email, and optionally first_name, last_name, personas (e.g. donor|volunteer or comma-separated), organization, jurisdiction (india, us, other). Valid persona codes match the system roles (e.g. foundation_admin, governance, donor, volunteer, …). New users get an unusable password until reset. Empty personas flags the row for assignment.
